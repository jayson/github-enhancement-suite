github-enhancement-suite
========================

Makes your pull requests look like this: http://i.imgur.com/EK8l2.png

* Inspired by Kastner's Last Lecture

Installation
1. Install Tampermonkey, Greasemonkey or Greasekit
2. Add this script
3. Reload github tabs
